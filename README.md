## 📥 Installation Instructions

### For Android/Mobile:
1. Download the `.mcpack` file from this repository
2. Open Minecraft Bedrock Edition
3. Go to **Settings** → **Global Resources**
4. Tap the downloaded `.mcpack` file
5. Activate the pack in **Create World**

### For Windows 10/11:
1. Download the `.mcpack` file
2. Open Minecraft Bedrock Edition
3. Navigate to **Settings** → **Global Resources**
4. Import the `.mcpack` file
5. Select it when creating a new world
